# JavaScript 1-2. 変数を使う ①

プログラミングでは JavaScript に関わらず変数という概念がある。
これは何かを一時的に保存しておく箱のようなもので、保存する名前は自由に決めてあとから何度でも呼び出すことが可能になる。

## 変数を定義する

```
const memo = "女の子は「NEWS」と言っているようだ。"
console.log(memo) // Consoleに上の内容が出力される
```

- `const 変数名 = 値`で定義することができる
- 定義した変数は、何度でも呼び出し可能
- 変数名は数字から始めたり、`-`を使うことはできない

## 変数を組み込む

```
const keyword = "NEWS"
console.log("女の子は「" + keyword + "」と言っているようだ。") // 結合子を使う
console.log(`女の子は「${keyword}」と言っているようだ。`) // テンプレートリテラルを使う
```

変数を文字列に組み込む方法は以下の 2 つ

- `"文字列" + 変数`のように結合する
- 文字列全体は` `` `（バッククォート）で囲み、変数は`${}`で囲う

## 数字を定義すれば演算可能

```
const num1 = 2
console.log(num1) // 2
```

```
const num2 = 3 + 4 // 計算結果を保存することも
console.log(num2 * 5) // 45
```

- 数字も変数に定義することができる
- 変数に保存された数字は演算可能
