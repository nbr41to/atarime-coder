# JavaScript 1-1. Hello World

「Hello World」とはプログラミングでコンピューターになにかを出力させるときによく使用される文字列である。
君が今イカに転生して「Hello World」な気持ちを JavaScript で表現してみよう。

## 文字列を Console に出力する

```
console.log("Hello World")
```

- 記号/符号は全て半角で
- `console.log()`の`()`の中が出力される
- 文字列は`""`（ダブルクォート）もしくは`''`（シングルクォート）で囲むことで表現可能
- 中身は日本語でも OK

## 数字を Console に出力する

```
console.log(123) // 123
console.log(3 + 5) // 8
console.log(3 - 5) // -2
console.log(3 * 5) // 15
console.log(3 / 5) // 0.6
```

- 数字はそのまま数字で表現可能
- 数字は半角で
- 演算をすることも可能
- 演算した場合は結果が出力される

## やってみよう
