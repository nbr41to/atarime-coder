# JavaScript 1-3. 変数を使う ②

プログラミングでは JavaScript に関わらず変数という概念がある。
これは何かを一時的に保存しておく箱のようなもので、保存する名前は自由に決めてあとから何度でも呼び出すことが可能になる。

JavaScript の変数には定義後に値を代入することができるものとそうでないものがある。

## 書き換え可能な変数を定義する

```
let score = 0
score = 800 // score に 800 を代入
```

- `let 変数名 = 値`で定義することができる
- `let`を使って定義した変数は`=`で何度でも代入することができる

## 数字の再代入の記法

```
let i = 0

// 全て同じ意味
i = i + 1
i += 1
i++

console.log(i) // 3
```

- `i = i + 1`は`i`に`i + 1`を再代入する
- 略記法がある
- 引き算も同じ感じでイケる
