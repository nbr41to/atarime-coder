# JavaScript 1-4. 条件分岐

「Hello World」とはプログラミングでコンピューターになにかを出力させるときによく使用される文字列である。
君が今イカに転生して「Hello World」な気持ちを JavaScript で表現してみよう。

## if 文で条件分岐

```
let score = 70;
if (score > 50) {
  console.log('イカしてる子');
}

score = 30;
if (score > 50) {
  console.log('イカしてる子');
} else {
  console.log('イカしてない子');
}
```

- `if(条件式){処理}`で条件を満たしたときに処理を実行
- `else`で条件を満たさなかったときに処理を実行
- JavaScript は上から順に処理が実行される

## 条件式について

```
let score = 30;
score > 50 // false
score >= 50 // false
score < 50 // true
score <= 50 // true
score === 50 // false
```

- 条件式は最終的に`true`か`false`という**真偽値**が返っている
- 数字の場合には不等式を扱うことができる
- `===`は値が等しいかどうかを判定する

## やってみよう 🦑

1. `direction`という変数名を`let`で定義し、`left`という文字列を定義しよう
2. その後`direction`が`left`であるか判定し、そうであれば `direction`に`west`を代入しよう（else は使用しない）
3. `direction`を Console に出力しよう
