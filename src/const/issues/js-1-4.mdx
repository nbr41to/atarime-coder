# JavaScript 1-4. 条件分岐

「Hello World」とはプログラミングでコンピューターになにかを出力させるときによく使用される文字列である。
君が今イカに転生して「Hello World」な気持ちを JavaScript で表現してみよう。

## if 文

```js
const score = 60;
if (score > 50) {
  console.log('あたりめ 🎯');
} else {
  console.log('するめ 🦑');
}
```

---

## やってみよう

1. `console.log()`を使って、「イカしてるぜ」と出力してみよう
2. `console.log()`を使って、「4 + 7」を出力してみよう
